# Assets Overview Page -- ZR Securities Trading App

> This prompt generates the multi-exchange portfolio dashboard with asset distribution and quick actions.

## Context
The assets page serves as the user's portfolio dashboard. Unlike OKX (single exchange), ZR must show holdings across multiple exchanges (HashKey, Bullish, OSL, VDX) with a unified total value.

**OKX Competitive Analysis Applied**: Adopted balance hide/show eye toggle (DA-010), asset list sorted by value, and quick action buttons. Enhanced with multi-exchange summary cards (DD-010), asset-class tabs (DD-001), and HKD default currency (DD-007).

## Interactions
- **Tap eye icon**: Toggle balance visibility (all values show "****")
- **Tap currency toggle**: Switch total value display between HKD/USD/BTC
- **Tap quick action (Deposit)**: Navigate to /deposit
- **Tap quick action (Withdraw)**: Navigate to /withdraw
- **Tap exchange card**: Filter asset list to that exchange only
- **Tap asset row**: Navigate to coin detail or trade page
- **Tap asset class tab**: Filter list by class
- **Pull-to-refresh**: Refresh portfolio values
