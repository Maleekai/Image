# Order Management Page -- ZR Securities Trading App

> This prompt generates the order management page with open/filled/cancelled tabs and cancel order flow.

## Context
Users need to monitor and manage their active and historical orders. This page provides a clear overview with the ability to cancel pending orders.

**OKX Competitive Analysis Applied**: Adopted OKX's tab-based order filtering (Open/Filled/Cancelled), order card layout pattern, and swipe-to-cancel gesture from flows 020_Orders (112-115). Enhanced with exchange badge on each order (DD-002) and ZR's 4-tab simplified navigation.

## Page Layout

### Zone 1: Status Bar + Nav Bar (88px)
- Status bar: 44px, same as all pages
- Nav bar: "Orders" title center (18px semibold), filter icon right (24px)

### Zone 2: Order Status Tabs (40px)
- Segmented control: `[Open] [Filled] [Cancelled]`
- Active: brand-blue background, white text, rounded-lg
- Inactive: transparent, text-secondary
- Container: bg-secondary, rounded-xl, mx-16, p-2
- Each shows count badge: "Open (3)" etc.

### Zone 3: Pair Filter (optional, 36px)
- "All Pairs" dropdown button, tapping shows bottom sheet with pair list
- When filtered: Shows selected pair name with X to clear

### Zone 4: Order List (remaining height, scrollable)
- Each order card: 96px height, bg-secondary, rounded-xl, mx-16, mb-8, p-16

**Order Card Layout**:
```
┌────────────────────────────────────────┐
│ BTC/USDT   [HashKey]    Limit Buy     │
│ Price: $67,234.50       Amount: 0.15  │
│ Filled: 0/0.15 BTC     Total: $10,085│
│ 2026-03-04 14:23  [─────────] [Cancel]│
└────────────────────────────────────────┘
```
- Row 1: Pair name (15px semibold) + Exchange badge + Order type/side (13px, green "Buy" / red "Sell")
- Row 2: Price (13px mono) + Amount (13px mono)
- Row 3: Fill progress bar (h-2, rounded, bg-success for filled portion) + Total
- Row 4: Date time (13px text-secondary) + Cancel button (for open orders only)

**Cancel button**: border border-danger, text-danger, rounded-lg, px-12, py-4, 13px
**Filled orders**: No cancel button, show fill price
**Cancelled orders**: Greyed out, show cancellation time

## Interactions
- **Tap tab**: Switch between Open/Filled/Cancelled with list animation
- **Tap Cancel button**: Show cancel confirmation bottom sheet
- **Confirm cancel**: Remove order from open list, add to cancelled with animation
- **Tap order card**: Expand to show full details (order ID, fees, timestamps)
- **Pull-to-refresh**: Refresh order list
- **Swipe left on order card**: Reveal cancel action (alternative to cancel button)
