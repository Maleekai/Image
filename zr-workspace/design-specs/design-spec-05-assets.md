## 2. Screen-by-Screen Analysis

### Screen 170-173: Assets Overview

**Purpose**: Portfolio dashboard showing all holdings, balances, and quick actions

**Layout Structure**:
- **Nav bar**: "Assets" title left, scan QR (left icon) + history clock (right) + settings gear (right)
- **Total Balance Card** (~140px):
  - "Total (BTC)" label with currency dropdown
  - Large balance: "0.00000000" in bold mono
  - Fiat equivalent: "≈ $0.00" secondary text
  - Eye icon to hide/show balance
- **Quick Actions Row** (80px): 4 circular buttons with labels
  - Deposit | Withdraw | Transfer | Convert
- **Account Tabs** (36px): "Trading | Funding | Earn | Trading bot"
- **Asset List**: Each coin row with icon + name + balance + fiat value
- **"Show all" toggle**: Include/exclude zero-balance assets

**OKX Asset Row Pattern**:
```
┌──────────────────────────────────────┐
│ [BTC Icon]  Bitcoin                  │
│              BTC                     │
│              0.00000000    $0.00     │
└──────────────────────────────────────┘
```

**Component Inventory**:

| Component | Type | Position | States | Notes |
|-----------|------|----------|--------|-------|
| Balance Card | Summary Card | Top | Visible / Hidden (eye toggle) | Large mono font balance |
| Currency Toggle | Dropdown | Balance card | BTC / USD / HKD / ETH | Changes denomination |
| Eye Toggle | Icon Button | Balance card | Visible / Hidden (shows ****) | Privacy feature |
| Quick Action Buttons | Icon + Label Grid | Below balance | Default / Pressed | 4 buttons, circular icons |
| Account Tabs | Segmented | Below actions | Trading / Funding / Earn / Bot | Filters asset list |
| Asset Row | List Item | Asset list | Default / Pressed | Icon + name + ticker + balance + value |
| Zero Balance Toggle | Switch | List footer | Show all / Hide zero | Filters zero-balance assets |

### Screen 174-178: Asset Detail & Sub-accounts

**Purpose**: Detailed view of specific asset with transaction history

**Observations**:
- Tap asset row -> Asset detail page
- Shows: Balance, Available, Frozen, In orders
- Transaction history below
- Quick actions: Trade, Deposit, Withdraw specific to that coin

## 5. ZR Adaptation

### Direct Adoption
- Balance card with eye toggle (privacy)
- Quick action row (Deposit/Withdraw/Transfer/Convert)
- Asset list with icon + name + balance + fiat value
- Zero-balance filter

### Modifications
- **Multi-Exchange Balances**: Show which exchange holds each asset via badge
- **Asset Class Tabs**: Replace "Trading/Funding/Earn/Bot" with "All/Crypto/Stocks/Forex/Commodities"
- **Exchange Summary Cards**: Show card per bound exchange with its total balance
- **Currency**: Default to HKD (Hong Kong Dollar) as fiat denomination
- **P&L Display**: Add daily P&L percentage and absolute value
- **SFC Compliance**: Show custody disclosure: "Assets held at [Exchange Name] under segregated custody"

### New for ZR
- **Multi-Exchange Summary** widget at top showing HashKey / Bullish / OSL / VDX split
- **Asset class breakdown** donut chart (Crypto 60%, Stocks 25%, Forex 15%)
- **Risk exposure** indicator (overall portfolio risk level)
